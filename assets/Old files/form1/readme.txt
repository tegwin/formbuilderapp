# Form Builder Setup Instructions

1. Upload the files into your server's `/form1/` directory.
2. Create a new MySQL database (e.g., form_builder).
3. Visit `http://localhost/form1/install/` in your browser.
4. Follow the step-by-step installer:
    - Requirements Check
    - Database Setup
    - Site Settings
    - Admin Account Setup
    - Final Installation
5. After installation:
    - Log in using the Admin account you created.
    - Start creating Forms and managing Submissions.
    - Configure Webhook URLs if needed.

Notes:
- Submissions are logged inside `/webhook_log.txt`.
- File uploads are stored in `/uploads/` folder.
- Admins can manage users, forms, settings through the Admin Panel.

Enjoy your Form Builder!
